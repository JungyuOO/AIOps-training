# storage_data_artifacts

Broad storage, object store, and model artifact cases.

Include PVC Pending, missing StorageClass, RWO multi-attach, node or PVC no-space errors, MinIO/S3 endpoint problems, bucket or prefix typos, 0B model objects, incomplete multipart uploads, missing CA bundles, and storage-initializer failures.

Evidence should name PVC/PV fields, events, bucket/prefix, object size, storage URI, initializer logs, and model artifact names.
